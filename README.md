## Objective

The objective of this project is to develop a Polynomial Regression model to predict employee salaries based on their position level. Since the relationship between position level and salary is nonlinear, Polynomial Regression is used to achieve better prediction accuracy than Linear Regression.

---

## Dataset

**Position Salaries Dataset**

**Kaggle Link:**  
https://www.kaggle.com/datasets/akram24/position-salaries

## Methodology

1. Import required libraries.
2. Load the dataset.
3. Display first five records.
4. Check dataset information.
5. Display summary statistics.
6. Check missing values.
7. Select input feature and target variable.
8. Split data into training and testing sets.
9. Apply Polynomial Features (Degree = 3).
10. Train Polynomial Regression model.
11. Predict salaries.
12. Evaluate using MAE, MSE, and R² Score.
13. Plot the Polynomial Regression curve.

---

## Results

- Successfully modeled the nonlinear relationship.
- Achieved good prediction accuracy.
- Polynomial Regression performed better than Linear Regression.

---

## Conclusion

Polynomial Regression is more suitable than Linear Regression for this dataset because it captures the nonlinear relationship between position level and salary, resulting in better prediction accuracy.
